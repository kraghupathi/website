#+title:  GSoC 2016 Student Instructions
#+SETUPFILE: ../org-templates/level-1.org

* A set of instructions for the students to enable them to apply to GSOC with Virtual Labs
  
  1. Join the virtual labs [[https://groups.google.com/forum/#!forum/gsoc-virtual-labs][mailing list]].
  2. Review our ideas page to see if you find a project that is interesting to you.
  3. Go through our [[http://vlabs-dev.vlabs.ac.in/community/][Community]] page to get started on the processes and get a
     peek at all the repositories on GitHub.
  4. You will need to create an account on GitHub.
  5. Write your application.  Please use this [[./template.org][template]].
  6. If you have any questions, please ask on the [[https://groups.google.com/forum/#!forum/gsoc-virtual-labs][mailing list]].
