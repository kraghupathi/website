#+SETUPFILE: ../org-templates/level-1.org

[[https://github.com/vlead/virtual-labs-as-a-service][Platform to support experiment as a service]] provides a platform to design an
experiment as a service.  The platform is a composition of javascript object
that facilitate capturing all the events on a experiment by a user.  These
events are exposed via REST API that are used by the outlier applications to
compute analytics, persistence, evaluation, etc. 

* Source Repository
   The sources are hosted on a [[https://github.com/vlead/virtual-labs-as-a-service][repository]] at GitHub.
